# API 
This is a simple `REST API` for managing `tasks`. 
## Resources
- `Task`

## Methods

- **GET**
    - `/tasks` - Get all tasks
    - `/tasks/{id}` - Get a task by id
- **POST**
    - `/tasks` - Create a new task and return the task id
- **PUT**
    - `/tasks/{id}` - Update a task by id
- **DELETE**
    - `/tasks/{id}` - Delete a task by id
